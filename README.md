Tower Defense - Estructuras de Datos en C++
El objetivo principal es demostrar la implementación y gestión de diferentes estructuras de datos lineales para controlar la lógica 
de torres, enemigos y oleadas.
Características:
El sistema simula un campo de batalla donde el usuario debe gestionar recursos y defensas para sobrevivir a oleadas de enemigos.
    Gestión de Torres: Inserción, eliminación y búsqueda de unidades defensivas.
    Sistema de Oleadas: Control cíclico de enemigos mediante listas circulares.
    Simulación por Turnos: Movimiento de enemigos y ataque de torres coordinado.
    Visualización en Consola: Renderizado del mapa y estado del juego en tiempo real.
Estructuras de Datos Implementadas:
El núcleo del proyecto se basa en la aplicación práctica de tres estructuras fundamentales:
    Lista Secuencial (Arreglos): Utilizada para la administración de las torres defensivas, permitiendo un acceso rápido y organizado.
    Lista Doblemente Enlazada: Implementada para gestionar los enemigos activos. Permite el recorrido en ambos sentidos, facilitando la
                                actualización de posiciones y la eliminación dinámica cuando un enemigo es derrotado.
    Lista Circular Enlazada: Diseñada para el control de las oleadas, permitiendo un avance continuo y la reutilización de la estructura 
                            durante la simulación.
Requisitos
    Compilador de DevC++ 
Funcionalidades principales
  - Registro, visualización y eliminación de torres  
  - Generación de oleadas de enemigos  
  - Simulación del juego por turnos  
  - Ataque automático de torres  
  - Sistema de vidas y economía (oro)  
  - Visualización del mapa en consola  
  - Menú interactivo  
Mecánica del juego
  - Los enemigos avanzan desde la posición 0 hasta la 20  
  - Las torres atacan automáticamente según su rango  
  - Si un enemigo llega al final → el jugador pierde una vida  
  - El juego termina cuando las vidas llegan a 0
